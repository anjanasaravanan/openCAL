Welcome to OpenCAL's documentation!
===================================

**OpenCAL** is an open-source Computed Axial Lithography printer. 

Check out the :doc:`background` section to learn about computed axial lithography.
Explore more work done by the Design for Nanomanufacturing lab `here <https://design-nano.berkeley.edu/>`_.

.. note::

   This project is under active development.
